# Fonts

- App font loading is configured in `nuxt.config.ts` with `@nuxt/fonts`.
- The project sans font token lives in `design-system/tokens.ts` as `font.family.sans`.
- `Montserrat` is loaded from Google Fonts with weights `400`, `500`, `600`, and `700`, normal style only, `latin` subset.
- Site-wide body and headings use Montserrat only (no second font family).
